# Wave 20 Delivery Report

Wave 20 extends the cumulative blueprint with a dedicated **hard-anatomy repair system** for faces, hands, feet, eyes, teeth, fingers, toes, and nails.

## Deliverables added
- strict crop/detail repair architecture
- region taxonomies for face, eye, mouth/teeth, hands, fingers, feet, toes, and nails
- pass profiles for crop repair and detail refine
- QA gates for local anatomy and global preservation
- schemas for contracts and evidence reports
- examples and runtime-bound source inventory
- local validation and scoring scripts

## Promotion boundary
This wave defines how hard-anatomy repair should work. It does not claim runtime promotion until real images, crops, masks, and evidence are generated and scored.
