# Organization System

This folder is the canonical organization layer for the full hyperrealism system.

## Subsystems

- `WAVE35_CANONICAL_STRUCTURE/` — local/repo/ComfyUI/App folder standards.
- `WAVE36_CATALOGS_INDEXES/` — catalogs, indexes, registry rules, search metadata.
- `WAVE37_MIGRATION_GOVERNANCE_HANDOFF/` — migration, validation, governance, release handoff.
- `GENERATED_INDEXES/` — generated project index outputs.
- `ORGANIZATION_VALIDATION/` — validation outputs and reports.
