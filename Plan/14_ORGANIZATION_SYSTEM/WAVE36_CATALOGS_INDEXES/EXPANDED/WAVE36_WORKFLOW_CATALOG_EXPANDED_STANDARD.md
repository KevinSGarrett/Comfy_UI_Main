# Wave 36 Workflow Catalog Expanded Standard

The workflow catalog tracks every ComfyUI/App Mode/orchestrator workflow.

## Required fields

- workflow_id
- workflow_name
- workflow_type
- engine_family
- canonical_status
- owner_domain
- source_path
- runtime_path
- app_mode_path
- required_models
- required_loras
- required_custom_nodes
- input_asset_classes
- output_prefixes
- preview_support
- final_render_support
- QA gates
- proof_status
- promotion_status
- archived_by
