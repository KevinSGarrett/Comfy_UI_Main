# Wave 36 Catalog Promotion Gate Model

Catalogs are part of release proof.

## Promotion requires

- file catalog refreshed
- workflow catalog refreshed
- asset catalog refreshed
- QA evidence catalog refreshed
- stale-index report pass
- registry search index pass
- release manifest updated

## Blockers

- uncataloged canonical file
- missing workflow entry
- missing asset entry
- missing QA evidence link
- stale release hash
- broken catalog path
