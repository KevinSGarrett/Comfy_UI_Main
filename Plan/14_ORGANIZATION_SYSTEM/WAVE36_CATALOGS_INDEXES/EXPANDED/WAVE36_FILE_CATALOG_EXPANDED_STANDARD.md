# Wave 36 File Catalog Expanded Standard

The file catalog should list every important file in the system.

## Required fields

- file_id
- path
- filename
- extension
- artifact_type
- owner_domain
- top_level_folder
- wave
- tags
- source_of_truth_role
- proof_status
- catalog_status
- size_bytes
- hash_when_available
- last_indexed
- archive_status

## Artifact type examples

```text
doc
schema
registry
example
script
workflow
manifest
qa_evidence
release_artifact
app_mode_config
ec2_config
generated_output
archive
```
