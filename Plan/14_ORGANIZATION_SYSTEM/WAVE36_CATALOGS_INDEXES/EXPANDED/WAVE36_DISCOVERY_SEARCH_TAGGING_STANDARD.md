# Wave 36 Discovery Search and Tagging Standard

## Required tags

- wave tag, for example `wave36`
- artifact type tag, for example `workflow`
- owner domain tag, for example `workflow_library`
- proof status tag, for example `proof_required`
- canonical status tag, for example `canonical` or `experiment`

## Recommended tags

- engine family
- workflow category
- output type
- QA domain
- App Mode app
- EC2 sync status
- archive status

## Search UX rule

A user should be able to find any important artifact by purpose, wave, engine, status, or owner folder.
