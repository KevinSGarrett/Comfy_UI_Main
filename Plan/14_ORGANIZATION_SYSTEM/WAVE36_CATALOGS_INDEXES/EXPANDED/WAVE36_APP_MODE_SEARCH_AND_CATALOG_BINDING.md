# Wave 36 App Mode Search and Catalog Binding

App Mode should not expose mystery controls.

## App Mode control catalog fields

- app_id
- control_id
- control_label
- workflow_input
- schema_field
- registry_entry
- preset_group
- validation_rule
- visible_to_user
- default_value
- allowed_values

## Rule

If a control is visible in App Mode, it must be searchable in the App Mode control catalog.
