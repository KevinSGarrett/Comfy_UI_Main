# Wave 36 Asset Catalog Standard

Assets include:
- models
- LoRAs
- VAEs
- ControlNets
- IPAdapter assets
- upscalers
- reference images
- masks
- control maps
- reference videos
- audio references

Each asset must have:
- asset_id
- type
- engine compatibility
- path
- size
- hash when available
- proof status
- owner folder
- archive status
