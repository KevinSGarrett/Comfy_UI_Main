# Wave 35 Generated Output Separation Policy

Generated outputs must be separated by purpose and quality level.

```text
07_GENERATED_OUTPUTS/
├── previews/
├── proxy_previews/
├── contact_sheets/
├── final_images/
├── video/
├── audio/
├── app_mode_outputs/
├── failed_outputs/
├── rejected/
└── archive/
```

## Output rule

No generated output should be treated as release-ready until a QA evidence record exists.
