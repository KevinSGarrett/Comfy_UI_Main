# Wave 37 Final Handoff Packet Expanded

## Final handoff must include

- local directory map
- repo structure map
- ComfyUI runtime map
- workflow library map
- App Mode map
- source-of-truth boundary matrix
- path alias registry
- generated catalogs
- stale-index report
- migration plan
- rollback plan
- local/EC2 sync runbook
- governance rules
- validation report
- release decision

## Handoff rule

A handoff is incomplete if the next person cannot answer:
- where files belong,
- which files are canonical,
- what is safe to edit,
- what is generated,
- what is runtime-only,
- what is release-certified,
- what proof is still required.
