# Wave 37 ComfyUI Runtime Handoff Packet

A ComfyUI handoff must describe exactly how the runtime is structured.

## Required handoff sections

- ComfyUI install path
- custom node list
- canonical workflow runtime path
- model folders
- LoRA folders
- reference input folders
- output folders
- preview output folders
- QA evidence folders
- required scripts
- known runtime boundaries
- blocked/unproven lanes

## Required proof

- workflow file exists
- required models listed
- required LoRAs listed
- outputs path declared
- QA evidence path declared
- runtime proof status recorded
