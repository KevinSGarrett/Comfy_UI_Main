# Wave 37 App Mode Handoff Packet

App Mode handoff converts complex graphs into tool-like user experiences.

## Required handoff sections

- app_id
- app purpose
- workflow mapped
- visible controls
- hidden controls
- preset groups
- profile groups
- input/output folders
- preview behavior
- final-render gating
- QA review controls
- release controls

## Rule

Every visible App Mode control must have a catalog entry and validation rule.
