# Wave 37 Governance Rules for Future Waves

Every future wave must include:

- AI PM task file
- current status file
- delivery report
- file index
- validation summary
- schemas, if new data structures are added
- examples, if schemas are added
- registries, if new categories are added
- validation script, if executable rules are added
- release note
- proof boundary statement

## Future wave rule

A wave that adds capability but does not add organization metadata is incomplete.
