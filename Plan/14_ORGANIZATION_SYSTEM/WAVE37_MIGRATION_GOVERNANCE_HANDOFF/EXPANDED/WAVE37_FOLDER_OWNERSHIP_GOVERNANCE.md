# Wave 37 Folder Ownership Governance

## Rule

Every top-level folder must have:

- owner domain
- allowed content
- blocked content
- catalog requirement
- source-of-truth role
- migration policy
- archive policy

## Change policy

No new top-level folder should be added unless:
- an existing folder cannot own the artifact,
- the new folder has a durable purpose,
- the folder is added to the directory registry,
- validation is updated.
