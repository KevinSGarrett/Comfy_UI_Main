# Wave 37 Local Directory Migration Plan

## Migration phases
1. Inventory current folders.
2. Map every folder to a canonical destination.
3. Move only after a manifest exists.
4. Create redirects or notes for legacy paths.
5. Validate no required file was lost.
6. Regenerate catalogs.
7. Run organization validation.

## Rule
Never bulk-move model/output folders without a manifest and rollback plan.
