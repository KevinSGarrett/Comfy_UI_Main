# Wave 37 Repo Cleanup and Monorepo Plan

## Cleanup goals
- keep source files in repo
- keep generated outputs out of repo
- keep heavy assets out of repo
- keep examples small
- keep schemas/registries versioned
- keep docs easy to browse

## Repo quality gates
- no unindexed scripts
- no unreferenced schemas
- no duplicate canonical workflows
- no generated cache folders
- no giant model files
