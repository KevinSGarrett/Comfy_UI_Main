# Wave 37 Organization Governance Rules

## Future wave rule
Every future wave must add:
- file index
- registry additions
- validation additions
- release notes
- source-of-truth statement
- proof boundary statement

## Folder rule
Every new top-level folder must be justified by ownership and purpose.

## Catalog rule
Every important artifact must be cataloged.

## Archive rule
Deprecated files should move to archive or be marked superseded, not left ambiguous.
