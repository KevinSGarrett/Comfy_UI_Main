# Wave 18 Video Surface-State Temporal Continuity Interface

Video/GIF support is in scope. This wave defines how surface realism carries across frames.

## Required temporal controls
- per-frame region ownership
- stable surface profile ids
- allowed intensity drift bands
- event-based transitions (for example, dry -> sweaty)
- frame repair hooks for flicker

## Temporal QA
Reject if adjacent frames show unexplained flicker in pore detail, cellulite strength, wetness, or fabric sheen.
