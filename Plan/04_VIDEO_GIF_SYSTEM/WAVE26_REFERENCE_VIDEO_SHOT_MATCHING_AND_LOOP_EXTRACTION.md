# Wave 26 Reference Video Shot Matching and Loop Extraction

## Full video reference
Use when the goal is to follow the timing and choreography of the source video.

## Segment reference
Use when only part of the video is needed.

## Loop extraction
If creating a GIF from a reference video, identify:
- start frame
- end frame
- seam compatibility
- loop type
- ping-pong option
- required stabilization or frame repair

## Shot matching
For video generation, map source-video phases to target shot phases:
- setup
- approach
- contact_start
- hold
- pressure_peak
- release
- recovery
