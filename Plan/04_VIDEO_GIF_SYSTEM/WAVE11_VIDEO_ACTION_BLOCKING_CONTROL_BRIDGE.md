# Wave 11 Video Action Blocking Control Bridge

## Bridge Principle

Image control maps can seed video, but video requires temporal contracts.

## Bridge Types

- still image keyframe to video;
- DWPose keyframe sequence;
- depth sequence;
- Canny/lineart keyframe sequence;
- camera path plus blocking plan;
- action timeline.

## Rule

A still image pose proof does not prove video pose control. Video requires its own runtime evidence.
