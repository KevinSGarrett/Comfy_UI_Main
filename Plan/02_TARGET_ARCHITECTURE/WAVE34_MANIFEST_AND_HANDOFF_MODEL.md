# Wave 34 Manifest and Handoff Model

## Release manifests
- file manifest
- source manifest
- workflow manifest
- App Mode manifest
- orchestrator manifest
- local proof manifest
- EC2 proof manifest
- QA certification manifest
- release decision manifest

## Final handoff packet
The handoff packet must tell the operator:
- what is included
- what is proven
- what is still proof-bound
- how to run validation
- how to run preview
- how to unlock EC2
- how to collect evidence
- how to block unsafe/incomplete release states
