# Wave 37 Expanded Migration Governance Handoff Architecture

Wave 37 adds operating process around the organized structure.

## Architecture layers
- migration manifests
- rollback manifests
- repo cleanup manifests
- ComfyUI handoff manifests
- App Mode handoff manifests
- local/EC2 sync manifests
- governance records
- organization validation reports
- final handoff packets

## Architecture rule
No structural change becomes canonical unless catalogs refresh and validation passes.
