# Wave 28A Micro-Motion Trigger Rules

## Breathing triggers
- always present at low amplitude for living characters unless deliberately still
- increases with exertion, fear/stress, recovery, or repeated action
- decreases during calm idle phases

## Tremble triggers
- fatigue high
- stress high
- unstable grip
- cold/shiver style state
- post-impact recovery

## Grip-change triggers
- contact graph pressure changes
- object weight shifts
- hand support needed
- fatigue affects hand stability

## Gaze/blink triggers
- emotional beat
- attention target changes
- character reaction
- camera close-up
- fatigue/stress increases blink/tremor probability

## Hair/fabric follow-through triggers
- head movement
- body acceleration
- wind/environment
- contact/release
- camera movement, if parallax should be visible

## Bounce/ripple/rebound triggers
- Wave 21 soft-body profile allows it
- Wave 22 contact graph or primary motion creates force
- Wave 23 deformation pass identifies compression/release
- Wave 27 frame QA can keep it stable
