# Wave 29 Body, Fatigue, and Repetition State Ledger

## Body state
Tracks posture, visible strain, surface state, compression history, and region continuity.

## Fatigue state
Tracks exertion and recovery over time.

## Repetition state
Tracks repeated actions so the system can introduce natural variation rather than repeating identical frames.

## Important
Fatigue should accumulate, transition, and recover. It should not randomly appear or reset between shots.
