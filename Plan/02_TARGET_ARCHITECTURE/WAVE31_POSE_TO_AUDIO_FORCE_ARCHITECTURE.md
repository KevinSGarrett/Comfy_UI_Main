# Wave 31 Pose-to-Audio Force Architecture

Wave 31 connects visual motion and contact to audio force.

## Core idea
Audio force should be derived from structured visual state, not prompt wording.

## Inputs
- pose timeline
- contact graph
- micro-motion events
- deformation/collision events
- action intensity
- motion velocity
- camera position and framing
- room/environment profile
- character/object depth order

## Outputs
- audio_force events
- spatial placement events
- reverb/room response selection
- gain/panning decisions
- QA targets and rerun policy
