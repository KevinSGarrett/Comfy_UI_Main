# Wave 19 Fabric Fold, Stretch, Cling, and Wrinkle System

Fabric realism is separated into behavior classes so the orchestrator can choose a controlled pass instead of using loose prompt wording.

## Fabric behavior classes
- loose drape
- normal fit
- tight stretch
- elastic compression
- wet/damp cling
- folded/bunched fabric
- wrinkle/crease field
- transparent/reflective/high-sheen material handling

## Inputs required
- clothing mask
- body-region adjacency map
- material profile
- tension direction
- contact/support owner
- pass intensity

## QA target
Fabric must follow body geometry and scene lighting. It must not erase the body silhouette, corrupt character identity, or change the approved pose.
