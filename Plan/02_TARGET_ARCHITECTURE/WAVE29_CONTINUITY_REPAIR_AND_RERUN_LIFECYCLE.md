# Wave 29 Continuity Repair and Rerun Lifecycle

## Repair ladder
1. local frame repair
2. local span repair
3. state ledger correction
4. rerun shot from prior approved keyframe
5. rerun whole scene segment
6. block promotion if continuity cannot be proven

## Common failure types
- clothing reset
- fatigue reset
- sweat/detail reset
- wrong character state
- repeated identical motion
- state jump across shots
- surface/contact state contradiction
