# Wave 29 Scene Progression and Clothing State Model

## Scene progression
Tracks story/sequence phase:
- establishing
- setup
- escalation
- peak
- recovery
- transition
- ending

## Clothing state
Tracks:
- clean/unchanged
- wrinkled
- stretched
- shifted
- compressed
- damp/sweaty
- partially displaced
- repaired/continuity locked

Clothing changes persist unless the scene explicitly resets or repair evidence justifies a correction.
