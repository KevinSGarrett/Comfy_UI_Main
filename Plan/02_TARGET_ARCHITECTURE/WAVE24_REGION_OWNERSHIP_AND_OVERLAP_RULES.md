# Wave 24 Region Ownership and Overlap Rules

## Region ownership map
Each character gets a region map for:
- head / hair / face
- torso
- arms / hands / fingers
- hips / legs / feet
- clothing / accessories
- contact regions
- protected identity regions

## Overlap classes
- clean separation
- contact overlap
- occlusion overlap
- support overlap
- invalid merge

Only contact, occlusion, and support overlaps are allowed. Invalid merge overlap blocks promotion.
