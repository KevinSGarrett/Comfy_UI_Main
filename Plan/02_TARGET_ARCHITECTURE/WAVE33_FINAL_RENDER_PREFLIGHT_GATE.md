# Wave 33 Final Render Preflight Gate

The final render preflight gate is the last checkpoint before expensive output.

## Required proof
- preview artifacts exist
- preview QA passed
- failed preview issues are repaired or accepted by revision
- selected take/variant is declared
- realism budget is justified
- compute budget is within policy
- EC2 hydration inputs are known
- output destination is declared
- rollback/stop condition is declared

## Rule
No final render may start from layout notes alone.
