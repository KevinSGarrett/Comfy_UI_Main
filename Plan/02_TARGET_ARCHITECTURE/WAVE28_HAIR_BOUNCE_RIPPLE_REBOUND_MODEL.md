# Wave 28 Hair, Bounce, Ripple, and Rebound Model

## Hair motion
Hair follows head movement, wind/environment, and body acceleration. Hair motion should lag slightly behind the primary motion.

## Bounce / ripple / jiggle
Bounce/ripple/rebound are secondary responses derived from Wave 21 soft-body profiles and Wave 22 contact graph intensity.

## Rule
Secondary motion must never overpower the primary motion. It is a supporting signal, not a new action.
