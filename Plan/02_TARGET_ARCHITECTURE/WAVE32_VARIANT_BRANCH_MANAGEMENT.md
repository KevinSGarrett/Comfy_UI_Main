# Wave 32 Variant Branch Management

Variants are controlled branches, not random alternate outputs.

## Variant examples
- camera angle variant
- lighting variant
- expression variant
- pose variant
- clothing state variant
- audio mix variant
- video motion variant
- detail pass variant

## Variant rules
- every variant has a parent plan or take
- every variant declares changed domains
- unchanged domains remain locked
- winning variants can become a promoted revision
- failed variants are kept as anti-patterns when useful
