# Wave 33 Realism Budget Model

The realism budget describes where detail matters and where it does not.

## Budget domains
- identity
- face/hands/feet/detail crops
- skin/material realism
- clothing/fabric realism
- contact/shadows/occlusion
- deformation/compression
- camera/lens/framing
- video temporal continuity
- audio/spatial audio sync
- final polish/upscale

## Budget levels
- draft
- preview
- proof
- final
- hero

## Rule
Hero-level detail is reserved for outputs that already passed preview QA.
