# Wave 33 Animatic and Shot Preview Plan

Animatics are low-cost representations of video/GIF/sequence intent.

## Animatic checks
- shot order
- camera cuts
- character placement
- pose progression
- motion readability
- contact timing
- occlusion order
- audio/event timing placeholders
- expected final render cost

## Output forms
- still keyframe strip
- low-resolution frame sequence
- contact sheet
- rough motion preview
- timeline JSON

## Rule
Final video rendering is blocked until animatic preview QA passes.
