# Wave 28A Micro-Motion Decision Engine

Wave 28A upgrades Wave 28 from a list of possible micro-motions into a decision system.

## Key answer

Reference videos are **not required** for micro-motion, but they are useful when exact timing, cadence, or body mechanics must follow a real source.

## Inputs that decide micro-motion
- scene phase
- action intensity
- contact graph pressure
- fatigue/exertion score
- character emotional state
- body state ledger
- clothing state ledger
- reference video timeline, if available
- user instruction priority
- safety bounds from identity/anatomy/contact QA

## Outputs
- breathing profile
- gaze/blink profile
- tremble profile
- grip-change profile
- hair/fabric follow-through profile
- bounce/ripple/rebound profile
- blocked/no-motion decision
