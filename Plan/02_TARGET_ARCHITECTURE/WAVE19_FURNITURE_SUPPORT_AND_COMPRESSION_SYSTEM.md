# Wave 19 Furniture Support and Compression System

Furniture and support surfaces must respond to body weight, body position, and contact area.

## Supported surfaces
- chair / seat
- couch / sofa
- bed / mattress
- pillow / cushion
- table / counter
- wall / floor / hard surface

## Compression types
- soft cushion indentation
- mattress depression
- pillow deformation
- fabric seat wrinkle
- hard-surface contact shadow without deformation
- body/furniture occlusion edge

## Core rule
A supported body must have believable contact points, visible support, and material-appropriate deformation. Soft materials compress; hard materials create contact shadow and occlusion instead of visible denting.
