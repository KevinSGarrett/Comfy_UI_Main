# Wave 33 Proxy Preview Tiers

## Tier 0 — Text / state preview
Checks plan completeness before any render.

## Tier 1 — Layout preview
Checks camera, composition, character count, and placement.

## Tier 2 — Still proxy
Low-step, low-resolution still output to check identity, pose, regions, masks, and major objects.

## Tier 3 — Contact sheet
Multiple low-cost candidates shown side-by-side for selection and rerun choice.

## Tier 4 — Animatic
Low-cost shot/keyframe sequence that checks timing, camera cuts, motion beats, and action readability.

## Tier 5 — Final preflight
A final gate that confirms preview QA, realism budget, and compute budget before EC2/final render is allowed.
