# Wave 25 Merge Prevention and Instance Separation

Merge prevention is the central QA goal of Wave 25.

## Blocked failure modes
- merged bodies
- merged hands/fingers
- merged limbs
- swapped faces/identities
- cloned or duplicated contact limbs
- prop fused into clothing/body
- hair blending between characters
- wrong-character mask edit

## Required evidence
- per-character masks
- region ownership map
- skeleton association
- depth order map
- before/after crop evidence for overlap zones
