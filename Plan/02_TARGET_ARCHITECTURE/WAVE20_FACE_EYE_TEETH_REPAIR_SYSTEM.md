# Wave 20 Face, Eye, and Teeth Repair System

## Face
- Preserve identity from Wave 08 character bible/reference pack.
- Repair only the failed facial feature unless full face structure is broken.
- Reject if identity, age, hairline, face shape, or expression drifts outside allowed variance.

## Eyes
- Validate symmetry, iris placement, pupil direction, catchlights, eyelids, and eyelashes.
- Reject mismatched gaze unless intentionally requested.

## Teeth and mouth
- Validate tooth count impression, spacing, gum/lip boundary, jaw/lip geometry, and mouth lighting.
- Reject melted teeth, duplicate rows, over-sharpened teeth, or pasted-looking smiles.
