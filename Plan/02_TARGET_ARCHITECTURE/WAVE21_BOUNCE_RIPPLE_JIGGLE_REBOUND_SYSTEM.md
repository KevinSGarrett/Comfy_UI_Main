# Wave 21 Bounce / Ripple / Jiggle / Rebound System

Bounce, ripple, jiggle, and rebound are temporal behaviors. In still images they can only be represented as static cues. In video/GIF, they require frame-to-frame evidence.

## Still-image representation
- contact compression
- surface displacement hints
- secondary fold/ripple detail
- plausible directional shape response

## Video representation
- tracked region mask
- stable material profile id
- frame-window evidence
- rebound timing
- damping behavior
- no identity, pose, or silhouette drift

## Promotion requirement
A video soft-body behavior is not promoted until temporal QA proves it does not flicker, teleport, or change material identity between frames.
