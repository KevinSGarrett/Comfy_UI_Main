# Wave 30 Impact and Action-Linked SFX

Action-linked SFX are generated from structured event data, not generic prompt wording.

## Event sources
- contact graph pressure changes
- object collisions
- prop interaction
- clothing/fabric tension
- soft-body compression/rebound
- footstep or body support events
- scene transitions

## Required event fields
- source_object
- target_object
- contact_type
- intensity
- duration
- material profile
- expected audio force
- sync frame/time
