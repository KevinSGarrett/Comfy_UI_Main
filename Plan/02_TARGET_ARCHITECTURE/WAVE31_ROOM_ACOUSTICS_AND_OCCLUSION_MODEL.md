# Wave 31 Room Acoustics and Occlusion Model

## Room acoustics
Room acoustics are determined from:
- room size
- wall/floor material
- furniture density
- open/closed environment
- camera/listener position

## Occlusion
Occlusion changes sound when the source is blocked by:
- body
- prop
- furniture
- wall/door
- clothing/fabric
- off-camera depth placement

## Occlusion effects
- lower gain
- low-pass filter
- muffled transient
- reduced clarity
- adjusted reverb tail
