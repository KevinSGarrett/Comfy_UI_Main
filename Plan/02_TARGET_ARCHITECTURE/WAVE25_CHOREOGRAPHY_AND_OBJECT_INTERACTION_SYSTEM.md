# Wave 25 Choreography and Object Interaction System

## Choreography phases
- setup
- approach
- contact_start
- hold
- pressure_peak
- release
- recovery

## Object interaction patterns
- one character holds object
- object between two characters
- object passes from one character to another
- character touches furniture/support object
- prop partially occludes a person

Object ownership must be explicit before any local repair pass is allowed.
