# Wave 23 Deformation Runtime Proof Lifecycle

Wave 23 is proof-bound until the AI system captures:

- input contact graph edge
- compiled deformation contract
- masks used by the pass
- ComfyUI workflow patch diff
- before/after crops
- preservation QA scores
- rerun reasoning if a pass fails

Only after repeated passing evidence on representative scenes may a deformation profile be promoted to runtime-proven.
