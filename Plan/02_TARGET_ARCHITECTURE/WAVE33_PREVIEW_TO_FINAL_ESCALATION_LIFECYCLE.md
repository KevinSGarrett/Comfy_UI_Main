# Wave 33 Preview-to-Final Escalation Lifecycle

## Lifecycle
1. plan preview
2. render low-cost proxy
3. run preview QA
4. repair preview failures
5. select take/variant
6. assign realism budget
7. assign compute budget
8. run final preflight
9. unlock EC2 final render
10. collect final evidence
11. run final QA

## Escalation principle
Preview success unlocks the next cost tier. Preview failure routes to preview repair, not final render.
