# Wave 29 Long-Form Continuity and Variation Architecture

Wave 29 gives the system a memory ledger across long-form sequences.

## Core purpose
Track what changed, why it changed, and whether it should persist.

## State domains
- character identity
- body state
- fatigue/exertion
- breathing
- clothing
- hair
- skin/surface
- contact/deformation
- props/furniture
- scene progression
- camera/framing
- repetition/variation

## Output
A continuity ledger that every shot, keyframe, frame repair, and rerun must read before promoting.
