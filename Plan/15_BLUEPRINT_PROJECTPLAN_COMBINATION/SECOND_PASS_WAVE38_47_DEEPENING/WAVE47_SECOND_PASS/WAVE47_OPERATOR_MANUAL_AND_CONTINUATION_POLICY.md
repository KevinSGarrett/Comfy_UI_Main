# Wave 47 Operator Manual and Continuation Policy

## Operator rule

Work from the combined operating model:

1. search the catalogs,
2. find the source requirement,
3. inspect the crosswalk,
4. find the backlog item,
5. check runtime proof boundary,
6. execute the next action,
7. validate,
8. update handoff/release notes.

## Future continuation policy

Future waves must extend the combined model, not create a separate plan.
