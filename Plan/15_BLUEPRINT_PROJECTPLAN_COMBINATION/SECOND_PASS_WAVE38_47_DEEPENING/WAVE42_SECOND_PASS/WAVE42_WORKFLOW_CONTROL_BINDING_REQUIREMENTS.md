# Wave 42 Workflow Control Binding Requirements

## Every workflow bridge must define

- workflow source path
- ComfyUI runtime path
- App Mode app
- visible controls
- hidden controls
- control schemas
- input folders
- output prefixes
- preview behavior
- final-render block
- QA evidence path

## Rule

A workflow cannot be considered combined with the blueprint plan unless its intended use, runtime path, and App Mode exposure are all mapped.
