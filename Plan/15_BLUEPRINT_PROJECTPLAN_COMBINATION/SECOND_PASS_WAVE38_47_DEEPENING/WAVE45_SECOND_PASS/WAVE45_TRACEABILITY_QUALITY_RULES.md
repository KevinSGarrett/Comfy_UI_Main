# Wave 45 Traceability Quality Rules

## Good traceability answers

- Which source document introduced this requirement?
- Which wave owns it?
- Which artifact implements it?
- Which registry/catalog indexes it?
- Which validation script checks it?
- Which evidence proves it?
- Which release decision promoted or blocked it?

## Trace quality levels

```text
level_0_untraced
level_1_source_only
level_2_source_to_wave
level_3_source_to_artifact
level_4_source_to_validation
level_5_source_to_evidence_and_release
```
