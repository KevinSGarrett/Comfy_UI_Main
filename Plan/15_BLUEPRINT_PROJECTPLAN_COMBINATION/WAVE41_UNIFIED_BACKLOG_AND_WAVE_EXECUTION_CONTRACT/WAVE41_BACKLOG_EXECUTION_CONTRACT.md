# Wave 41 Unified Backlog and Wave Execution Contract

## Backlog merge rule
The blueprint/manual/project plan becomes the planning backlog.  
The cumulative wave system becomes the implementation backlog.  
The tracker/reality CSVs become the evidence and status backlog.

## Every backlog item must have
- backlog_id
- source_layer
- related_wave
- owner_domain
- implementation_target
- proof_required
- current_status
- next_action
- blocked_reason, if any
