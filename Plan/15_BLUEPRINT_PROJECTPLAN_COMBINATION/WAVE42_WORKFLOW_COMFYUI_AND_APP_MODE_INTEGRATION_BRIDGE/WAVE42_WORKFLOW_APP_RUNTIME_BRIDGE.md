# Wave 42 Workflow, ComfyUI, and App Mode Integration Bridge

## Bridge fields
- workflow_id
- ComfyUI runtime path
- workflow library source path
- App Mode app
- visible controls
- hidden controls
- input assets
- output prefixes
- pruntime gate
- final render gate
- QA evidence path
- release proof path

## Rule
A workflow is not App Mode ready unless its controls, presets, inputs, outputs, and proof gates are cataloged.
