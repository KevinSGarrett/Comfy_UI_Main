# Wave 21 Firmness / Softness / Compression Pass Plan

## Firmness correction
Use when a region looks too inflated, too melted, or too weakly structured.

## Softness correction
Use when a region looks plastic, rigid, or disconnected from contact/support.

## Compression correction
Use only when there is declared contact or support context.

## Denoise ranges
- subtle firmness/softness rebalance: 0.10-0.22
- contact compression: 0.18-0.32
- furniture/support compression: 0.18-0.35
- large sag correction: 0.20-0.35
