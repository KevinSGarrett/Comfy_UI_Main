# Wave 21 Image Soft-Body Material Pass Plan

## Objective
Use material profiles to create controlled still-image soft-body cues.

## Still-image cues
- pressure/compression shape cues
- support surface indentation
- gravity-aware sag
- skin/fabric/furniture response
- mild ripple or fold cues

## Pass order
1. approved base image
2. approved frame/body/hard-anatomy state
3. mask ownership
4. material profile selection
5. low-denoise soft-body pass
6. QA scoring
7. cleanup or promotion
