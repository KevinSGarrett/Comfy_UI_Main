# Wave 25 Object/Prop Interaction Repair Plan

Object interaction is treated as a first-class event.

## Examples of object states
- held_by_character_A
- held_by_character_B
- between_characters
- occluding_character
- supported_by_furniture
- resting_on_body_region

A prop cannot be repaired unless its owner, depth order, and contact region are known.
