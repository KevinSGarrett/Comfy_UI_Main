# Wave 18 Surface Texture and Fabric State Plan

Wave 18 treats skin and fabric together when they directly interact.

## Fabric states
- dry matte
- soft cotton / diffuse
- elastic compression / stretched
- satin / silky sheen
- damp / clinging
- glossy / wet

## Cross-boundary rule
At every skin/clothing boundary, the system must preserve:
- seam continuity
- contact-shadow continuity
- wrinkle direction
- body-pressure logic
