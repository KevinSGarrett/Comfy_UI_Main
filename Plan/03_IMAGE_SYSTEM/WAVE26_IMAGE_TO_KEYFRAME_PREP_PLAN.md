# Wave 26 Image-to-Keyframe Prep Plan

Before temporal generation begins, the system prepares the still-image assets.

## Prep steps
1. Select approved still anchor(s).
2. Confirm identity and framing continuity.
3. Export or derive pose/depth/control map references.
4. Export or derive editable/protected/contact masks.
5. Label each still with its timeline role.
6. Build the first temporal plan.

## Important rule
Do not start temporal generation from a still that still needs major anatomy or composition correction.
