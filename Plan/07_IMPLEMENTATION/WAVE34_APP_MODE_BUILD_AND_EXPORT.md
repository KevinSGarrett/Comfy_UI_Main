# Wave 34 App Mode Build and Export

## App Mode export requirements
- Every user-facing control maps to a schema field.
- Controls must avoid exposing raw low-level graph hazards.
- Engine/model family compatibility must be enforced.
- Final render cannot be triggered directly.
- Preview QA and final preflight must be visible to the operator.
- Runtime status must show proof-bound lanes clearly.

## App Mode release output
The export should produce:
- App Mode workflow file
- App Mode control manifest
- App Mode QA report
- App Mode release decision
