# Wave 21 Implementation Manual

## Build sequence
1. Start from the latest approved image or frame.
2. Select an owned region mask from Wave 13.
3. Select a soft-body material profile.
4. Confirm contact/support context if compression is requested.
5. Compile a deformation pass plan.
6. Patch the low-denoise refine workflow.
7. Run the pass and collect evidence.
8. Score profile match and preservation.
9. Promote, rerun, fallback, or stop.
