# Wave 13 — Mask Storage and Git Policy

## Commit to Git

- Schemas
- Contracts
- Small examples
- Registry files
- Test fixtures under size limits

## Do not commit by default

- Large generated masks
- Output images
- Debug previews
- Video frames
- Full runtime evidence bundles

## Preferred runtime storage

Use local runtime folders during development and S3 or artifact storage for large reproducible evidence bundles.
