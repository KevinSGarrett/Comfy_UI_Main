# Comfy_UI_Main

Structured local/GitHub repository for the ultra-hyperrealism ComfyUI image, video, GIF, and audio generation system.

## Local root

```text
C:\Comfy_UI_Main
```

## What belongs in this repo

```text
docs
workflow templates
schemas
orchestration code
QA scripts
manifests
source inventories
App Mode specs
tests
```

## What does not belong in this repo

```text
model binaries
large LoRA/checkpoint files
generated renders
EC2 sync bundles
raw output folders
large audio/video files
```

## Runtime rule

Develop locally first. Turn on EC2 only through the runtime proof gate.
