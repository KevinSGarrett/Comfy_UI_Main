# Wave 19 Audio Fabric/Prop/Furniture Foley Interface

Wave 19 exports metadata that the audio layer can later use for foley.

## Foley metadata examples
- fabric rustle intensity
- tight fabric stretch/slide
- object pickup/setdown
- furniture creak
- cushion/mattress movement
- body shifting against support surface

Audio generation remains proof-bound until the audio runtime lane is wired and QA-proven.
