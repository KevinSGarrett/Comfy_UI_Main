# Wave 12 Audio/Visual Frame Boundary

Wave 12 is primarily visual, but it still matters for audio/video because shot continuity affects dialogue, Foley, and voice assignment.

## Audio-linked composition checks

- If a character speaks, the visible speaker assignment should match the audio plan when the shot shows the speaker.
- If the shot is a close-up, audio timing should not imply off-screen action that the frame plan cannot support.
- If video contains multiple characters, persistent visual IDs should map to persistent voice IDs.

This file does not define audio generation. It defines the boundary where frame composition metadata supports downstream audio/video synchronization.
