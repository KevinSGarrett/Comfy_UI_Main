# Wave 30 Breathing, Foley, and SFX Plan

## Breathing
Generated from fatigue/exertion and visual breathing state.

## Foley
Generated from motion/contact/material/prop state.

## SFX
Generated from declared action events.

## Required binding
Every breath/foley/SFX item must point to:
- scene_id
- shot_id
- segment_id
- event_id
- timecode/frame range
- visual source event
- QA target
