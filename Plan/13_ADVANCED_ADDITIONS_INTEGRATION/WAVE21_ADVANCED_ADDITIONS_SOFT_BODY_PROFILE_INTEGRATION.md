# Wave 21 Advanced Additions Soft-Body Profile Integration

Advanced additions that affect soft-body behavior must register as material/profile metadata instead of bypassing the profile system.

## Integration points
- material profile axis values
- region binding defaults
- compression/rebound profiles
- video motion profiles
- QA scoring dimensions
- App Mode controls

Any future advanced soft-body module must output profile/evidence data compatible with Wave 21 schemas.
