# Wave 23 Deformation, Collision, and Indentation QA Gates

A Wave 23 pass fails if any of the following are true:

1. source or target ownership is ambiguous,
2. a required mask is missing or misaligned,
3. anatomy reads as merged or chopped,
4. contact boundary looks painted rather than physical,
5. deformation extends beyond allowed scope,
6. nearby texture continuity is lost,
7. fingers/hands become malformed,
8. framing or identity drifts,
9. temporal rebound logic is missing when the media type requires it.
