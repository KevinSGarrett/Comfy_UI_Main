# Wave 25 Multi-Character Interaction QA Gates

A multi-character interaction fails if:
1. any character lacks an instance id,
2. masks overlap without declared occlusion,
3. source/target ownership is ambiguous,
4. depth order contradicts contact shadow,
5. limbs or props merge,
6. identity swaps occur,
7. wrong-character edit occurs,
8. object ownership is unclear,
9. choreography phase is inconsistent,
10. frame placement drifts.
