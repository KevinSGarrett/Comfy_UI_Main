# Wave 10 Camera Framing QA Gates

## Gate 1 — Plan Completeness

Pass only if the camera plan has:

- shot size
- lens profile
- camera angle
- framing profile
- depth plan
- subject slots
- QA goals

## Gate 2 — Crop Correctness

Pass only if:

- full-body shots do not cut off required body parts
- half-body shots crop intentionally
- close-ups preserve identity anchors
- detail inserts include enough context

## Gate 3 — Focus and Depth

Pass only if:

- focus target is sharp
- depth of field matches plan
- background blur does not hide required evidence
- multi-character subjects remain readable

## Gate 4 — Multi-Character Composition

Pass only if:

- subject count is correct
- identities do not merge
- screen positions match plan
- depth order matches plan
- occlusion policy is respected

## Gate 5 — Promotion

Blocked unless camera/framing evidence is attached to the output manifest.
