# Wave 13 — Mask Evidence Manifest Requirements

Each mask evidence manifest must include:

- mask ID,
- target type,
- owner ID,
- scale,
- source image,
- mask path,
- dimensions,
- coverage percentage,
- edge score,
- overlap report,
- before/after output evidence,
- SHA256 or equivalent fingerprint.

Promotion is blocked if the evidence manifest is missing or cannot be parsed.
