# Wave 33 Proxy Preview QA Gates

Preview QA checks:
- plan completeness
- character count
- identity lock
- frame composition
- pose readability
- mask/region ownership
- contact/occlusion readability
- camera/framing
- shot timing, if sequence
- major audio event placeholders, if sequence/audio
- expected final cost tier
- rerun recommendation

Preview passes only when it is good enough to justify the next cost tier.
