# Wave 19 Prop/Furniture Support Tests

## Prop tests
- held object with correct grip
- object resting on table/surface
- prop touching clothing/body
- accessory attached to body/clothing

## Furniture tests
- sitting on soft cushion
- lying on mattress/pillow
- leaning on hard surface
- body/furniture occlusion at contact boundary

Every test must include no-floating, no-clipping, shadow, and support-state scores.
