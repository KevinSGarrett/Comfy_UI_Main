# Wave 30 Audio QA Gates

An audio output cannot promote unless all checks pass:

- audio files exist
- files are decodable
- duration matches manifest
- loudness is in allowed range
- no clipping
- no silent required event
- dialogue is assigned to correct character
- breathing matches exertion/fatigue
- foley/SFX sync to visual event
- ambience/music are not masking required sounds
- final mix has QA evidence
