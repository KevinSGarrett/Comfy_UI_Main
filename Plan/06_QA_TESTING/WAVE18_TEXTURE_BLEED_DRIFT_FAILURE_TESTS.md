# Wave 18 Texture Bleed and Drift Failure Tests

## Failure types
- texture bleed outside mask
- pore over-sharpening or noisy artifacting
- cellulite converted into body-shape distortion
- global wet/oily look when only local moisture was requested
- pressure marks without real contact geometry
- clothing seam breakage
- identity drift from aggressive refine

Every failure must be recorded and mapped to a rerun policy.
