# Pulled Back Artifacts

Codex Desktop places EC2 pullback evidence here after GPU runtime validation.

Expected subfolders per run:

```text
<run_id>/
  logs/
  reports/
  images/
  videos/
  audio/
  workflows/
  PULLBACK_RECORD.json
```

Large generated outputs should not be committed to GitHub unless intentionally selected as small QA evidence. Keep registry/manifest metadata in Git, not bulk media.
