# Wave 61 Scope

## Name
Wave 61 — Strict Autonomous QA, Testing, Visual Review, Audio Review, and Video Review System

## Goal
Create the strict QA and review framework that Codex must use before it can claim any image, video, audio, workflow, script, model, prompt, LoRA, ComfyUI graph, or generated artifact is complete.

## Required result
The pack must provide:

- QA master protocol
- image review protocol
- video review protocol
- audio review protocol
- multimodal scorecard
- ComfyUI workflow testing protocol
- model / LoRA / checkpoint validation protocol
- prompt / negative prompt QA protocol
- failure classification and retest protocol
- done certification evidence protocol
- supporting templates, schemas, evidence guidance, and helper scripts

## Packaging note
This wave defines the QA framework and artifacts to use during runtime work. It does not certify the entire project as already QA-complete.
