# QA Evidence Directory Guidance

Store QA evidence records, screenshots, logs, summaries, and certification records here or in a linked artifact directory.

Recommended structure:

- `Image_Reviews/`
- `Video_Reviews/`
- `Audio_Reviews/`
- `Workflow_Validation/`
- `Model_Validation/`
- `Done_Certifications/`

Codex must preserve traceability between this evidence and the tracker/itemized list rows.
