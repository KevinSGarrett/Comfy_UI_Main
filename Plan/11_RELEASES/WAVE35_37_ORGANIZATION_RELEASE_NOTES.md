# Waves 35–37 Organization Release Notes

## Wave 35 — Canonical System Structure
Defines canonical local, repo, ComfyUI, App Mode, workflow, naming, versioning, and source-of-truth structure.

## Wave 36 — Catalogs and Indexes
Adds generated project index, file catalog, workflow catalog seed, script catalog, catalog schemas, examples, and validation policy.

## Wave 37 — Migration, Governance, and Handoff
Adds migration plans, local/EC2 sync structure, governance rules, organization validation, and final handoff documentation.

## Release purpose
This release is intended to make the system easy to navigate, maintain, validate, migrate, and extend without losing track of assets, workflows, proof files, or release gates.
