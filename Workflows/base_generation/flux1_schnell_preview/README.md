# FLUX.1 Schnell Preview

This lane is the generic four-step, 512 x 512 local preview route for the existing
`flux1-schnell-fp8.safetensors` checkpoint. It extends the historical
`true_flux_schnell_reference_smoke` topology with current model hashes, live
ComfyUI history, pulled-back output, technical QA, and direct visual QA.

It is not a substitute for the FLUX.1 dev production lane and does not establish
robustness beyond the exact bounded prompt, seed, resolution, and runtime proof.
